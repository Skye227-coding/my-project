## Notice
### git相关
#### 连接自己的本地项目带github：https://www.jianshu.com/p/6de75cba5c8a
#### 切换远程仓库地址办法：
### 在项目目录的命令行下输入以下代码
```
git remote set-url origin xxxxaddressxxx.git
```
```
git push
```
如果先连接了github又切换到码云，再想切换回来更新项目可以用以上指令
### 项目启动
前端：
```
npm run serve
```
后端：
```
nodemon  server.js
```
数据库：不可以忘记打sudo
```
sudo mongod
```
另外起一个命令行窗口
```
mongo
```
